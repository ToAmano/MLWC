# src/cmdline/

This directory contains command-line tools for various tasks, including training, testing, and applying machine learning models.

## Description

This directory provides a set of command-line scripts for performing various tasks related to machine learning models. These tools are designed to be used from the command line and provide functionalities for training models, testing models, making predictions, and other related tasks.

## Scripts

- `CPtrain.py`: A command-line tool for training, testing, and predicting using machine learning models. It supports subcommands for training, testing, prediction, influence graph analysis, PCA, and generating sample input files.
- `CPextract.py`: A command-line tool for extracting data from CP (Car-Parrinello) simulations.
- `CPmake.py`: A command-line tool for generating input files for CP simulations.
- `CPml.py`: A command-line tool for machine learning related tasks.
- `CPdescripter.py`: A command-line tool for generating descriptors for machine learning models.
- `ml_predict.py`: A command-line tool for making predictions using machine learning models.

## Subdirectories

- `cpextract_cpmd/`: Contains scripts related to extracting data from CPMD simulations.
- `cpextract_diel/`: Contains scripts related to extracting dielectric properties.
- `cpmake/`: Contains scripts related to generating input files.
- `cptrain/`: Contains scripts related to training machine learning models.
- `cptrain_pred/`: Contains scripts related to prediction using trained models.
- `cptrain_test/`: Contains scripts related to testing trained models.
- `cptrain_train/`: Contains scripts related to training machine learning models.
